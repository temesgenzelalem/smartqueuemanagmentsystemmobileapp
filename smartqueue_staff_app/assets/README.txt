Place the Tsehay Bank logo image here with the filename `tsehay_logo.png`.

Recommended sizes:
- Launcher/icon source: 1024x1024 PNG
- Header display: at least 200x200 PNG (will be scaled)

After adding the file, run:

flutter pub get
flutter pub run flutter_launcher_icons:main  # if using launcher icon tool

Or rebuild the app to pick up the new asset.
